# Spark-Github项目预测系统看板配置指南
## 1. 文档说明
### 1.1 适用场景
通过**CSV文件导入**配置「Spark-Github新生代优质项目早期预测系统看板」，覆盖CSV字段映射、图表配置、看板组装全流程，匹配你的CSV字段结构。
### 1.2 前置条件
- 已部署DataEase（≥v2.0）并登录；
- 已准备好包含完整字段的项目数据CSV文件；
- 具备DataEase看板编辑权限。


## 2. CSV文件导入与数据集配置
### 2.1 CSV文件导入（数据源）
#### 操作路径
左侧栏「数据源」→「新建数据源」→「文件数据源」→「上传文件」

#### 配置步骤
1. 上传你的项目数据CSV文件；
2. 编码选择`UTF-8`，分隔符选择`逗号`；
3. 数据源名称填写：`Spark-Github项目全量数据（CSV）`，点击「保存」。

### 2.2 数据集字段映射（自动识别）
DataEase会自动识别CSV字段，映射结果如下：

| CSV原始字段                | DataEase字段名           | 字段类型   |
|---------------------------|--------------------------|------------|
| full_name                 | 仓库全名                 | 字符串     |
| language                  | 开发语言                 | 字符串     |
| stargazers_count          | 星标数                   | 数值       |
| created_at                | 创建时间                 | 时间       |
| community_vitality_score  | 社区活力度得分           | 数值       |
| code_health_score         | 代码健康得分             | 数值       |
| maintenance_efficiency_score | 维护效率得分         | 数值       |
| topic_innovation_score    | 主题创新得分             | 数值       |
| external_appeal_score     | 外部吸引力得分           | 数值       |
| main_topic                | 主主题                   | 字符串     |
| overall_potential_score   | 综合潜力得分             | 数值       |
| overall_potential_level   | 综合潜力等级             | 字符串     |


## 3. 可视化图表配置（匹配看板组件）
### 3.1 核心指标卡片

| 卡片名称       | 配置项                     | 取值                          |
|----------------|----------------------------|-------------------------------|
| 项目总数       | 字段+聚合方式              | 仓库全名 + 计数               |
| 平均潜力分     | 字段+聚合方式              | 综合潜力得分 + 平均值         |
| 涉及语言数     | 字段+聚合方式              | 开发语言 + 去重计数           |
| 高潜力项目数   | 字段+聚合方式+筛选         | 综合潜力等级 + 计数 + 筛选“综合潜力等级=高” |

### 3.2 潜力等级分布环形图

| 配置项       | 取值                          |
|--------------|-------------------------------|
| 分类字段     | 综合潜力等级                  |
| 数值字段     | 仓库全名                      |
| 聚合方式     | 计数                          |
| 图表标题     | 潜力等级分布环形图            |
| 图例说明     | 综合潜力等级（高/中/一般）    |

### 3.3 语言占比Top10饼图

| 配置项       | 取值                          |
|--------------|-------------------------------|
| 分类字段     | 开发语言                      |
| 数值字段     | 仓库全名                      |
| 聚合方式     | 计数                          |
| 筛选条件     | 按“计数”降序取前10            |
| 图表标题     | 语言占比Top10饼图             |

### 3.4 潜力度与外部吸引力回归关系图

| 配置项       | 取值                          |
|--------------|-------------------------------|
| 图表类型     | 折线图                        |
| X轴字段      | 社区活力度得分                |
| Y轴字段      | 综合潜力得分                  |
| 数据系列     | 综合潜力得分（橙色折线）      |
| 图表标题     | 潜力度与外部吸引力回归关系图  |

### 3.5 高潜力项目主题词云图

| 配置项       | 取值                          |
|--------------|-------------------------------|
| 词云字段     | 主主题                        |
| 权重字段     | 综合潜力得分                  |
| 筛选条件     | 综合潜力等级=高               |
| 图表标题     | 高潜力项目主题词云图          |

### 3.6 潜力值排行榜

| 配置项       | 取值                          |
|--------------|-------------------------------|
| 表格字段     | 仓库全名、综合潜力得分        |
| 排序方式     | 综合潜力得分（降序）          |
| 表格标题     | 潜力值排行榜                  |


## 4. 看板组装配置
按以下布局拖拽组件：
1. **顶部区域**：4个核心指标卡片（项目总数、平均潜力分、涉及语言数、高潜力项目数）；
2. **中间区域**：左侧（潜力等级分布环形图）、中间（语言占比Top10饼图）、右侧（AI助手组件）；
3. **底部区域**：左侧（潜力度与外部吸引力回归关系图）、中间（高潜力项目主题词云图）、右侧（潜力值排行榜）。
